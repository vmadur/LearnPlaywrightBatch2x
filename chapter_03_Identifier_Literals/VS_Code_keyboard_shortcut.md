# VS Code Keyboard Shortcuts - Windows

## Window Management
| Action | Shortcut |
|---|---|
| New Window | `Ctrl+Shift+N` |
| Close Window | `Alt+F4` |
| Close Editor | `Ctrl+F4` |
| Split Editor | `Ctrl+\` |
| Focus Editor Group 1 | `Ctrl+1` |
| Focus Editor Group 2 | `Ctrl+2` |
| Focus Editor Group 3 | `Ctrl+3` |
| Focus Next Editor Group | `Ctrl+K Ctrl+Right` |
| Focus Previous Editor Group | `Ctrl+K Ctrl+Left` |
| Toggle Sidebar | `Ctrl+B` |
| Toggle Panel | `Ctrl+J` |
| Toggle Terminal | `` Ctrl+` `` |
| Toggle Full Screen | `F11` |
| Toggle Zen Mode | `Ctrl+K Z` |

## Command Palette & Settings
| Action | Shortcut |
|---|---|
| Command Palette | `Ctrl+Shift+P` |
| Open Settings | `Ctrl+,` |
| Open Keyboard Shortcuts | `Ctrl+K Ctrl+S` |

## Tips
- Shortcuts with two key combinations mean: Press the first combination, then the second (e.g., `Ctrl+K Ctrl+S` means hold Ctrl+K, release, then hold Ctrl+S)
- You can customize all shortcuts in: **File > Preferences > Keyboard Shortcuts** (`Ctrl+K Ctrl+S`)
- Search for any command in the Command Palette using `Ctrl+Shift+P`
