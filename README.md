# Learn Playwright - Batch 2

A JavaScript fundamentals curriculum designed to build a strong foundation for Playwright test automation.

## Course Content

| Chapter | Topic | Files |
|---------|-------|-------|
| 01 | Basics | Hello World, Node.js setup, hot code |
| 02 | JavaScript Concepts | Variable declaration and reassignment |
| 03 | Identifier & Literals | Identifier rules, naming conventions, Unicode, comments, VS Code shortcuts |
| 04 | JavaScript Concepts | `var`/`let`/`const`, scoping, hoisting |

## Prerequisites

- Node.js (v24+)
- VS Code
