# Learn Playwright - Batch 2

A JavaScript fundamentals curriculum designed to build a strong foundation for Playwright test automation.

## Course Content

| Chapter | Topic | Files |
|---------|-------|-------|
| 01 | Basics | Hello World, Node.js setup, verify setup, hot code |
| 02 | JavaScript Concepts | Variable declaration and reassignment |
| 03 | Identifier & Literals | Identifier rules, naming conventions, Unicode, comments, VS Code shortcuts |
| 04 | JavaScript Concepts | `var`/`let`/`const`, functions, scoping, hoisting, temporal dead zone |
| 05 | Literals | Numeric, string, template literals, backtick, null, undefined |
| 06 | Operators | Arithmetic, modulus, exponentiation, comparison (strict/loose), logical, ternary, type, increment/decrement, nullish coalescing |
| 07 | If-Else | Conditional branching, if/else-if/else, API status checks, grade calculator, leap year |
| 08 | Switch Statement | Multi-way branching, default case, no-break behavior, switch grouping, real-time examples |
| 09 | User Input | Node.js readline, synchronous prompt, CLI user input |

## Prerequisites

- Node.js (v24+)
- VS Code
